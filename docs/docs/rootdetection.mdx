Root detection is mostly deferred to RootBeer. The configuration looks like this:

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
    defaultValue="default"
    values={[
        { label: 'Safe to run (Default)', value: 'default', },
        { label: 'Reporting', value: 'reporting', },
    ]
    }>

<TabItem value="default">

```kotlin
safeToRun(buildSafeToRunCheckList {
    add {
        rootDetectionCheck()
    }
})
```

</TabItem>

<TabItem value="reporting">

```kotlin
rootDetection {
    tolerateRoot = false
    tolerateBusyBox = true
}
```

We also have an additional option (see blacklisting apps for more information) to deny running if we are able to see
blacklisted apps

For example:

```kotlin
 blacklistConfiguration {
    +"com.abc.def"
    blacklistRootingApps()
}.error()
```

</TabItem>
</Tabs>
